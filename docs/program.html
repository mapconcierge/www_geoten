3.株式会社MIERUNE
http://www.mierune.co.jp/

MIERUNE地図として、OpenStreetMapを独自スタイルで提供しており、
1．低コストでの導入が可能
2．印刷などへの二次利用が可能
3．さまざまなデザインに対応
といった特徴を持っています。
OpenStreetMapやweb地図の活用、地図データの配信について展示します。
また、MIERUNEは位置情報可視化・解析に広く使われているオープンソースデスクトップGIS「QGIS」のスポンサーとなっています。
QGISの利活用、プラグイン開発についても紹介します。
